## Installation

To install and run this project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/RonMizrahi/golang-firstservice.git
    ```

2. Change to the project directory:
    ```sh
    cd golang-firstservice
    ```

3. Build the project:
    ```sh
    go build
    ```

4. Run the project:
    ```sh
    ./golang-firstservice
    ```
